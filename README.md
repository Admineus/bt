# BITREVERSE - REVERSE MINING
Bitcoin, Altcoins and Ethereum address collision mining software.
Multi GPUs, full CUDA optimised.

## Build dependencies

Windows:
  - Visual Studio 2015
  - CUDA 9.2 Toolkit
  - Boost

Linux:
  - nvcc (CUDA 9.2) see env.sh and compile.sh
  - CUDA 9.2 Toolkit
  - Boost
